Duration

Модуль определяет тип [`Duration`] — компактное и точное представление
временного интервала с наносекундной точностью.

1. Общая идея

[`Duration`] хранит длительность как знаковое 64-битное целое число,
представляющее количество наносекунд. Это позволяет описывать как
положительные, так и отрицательные интервалы времени в диапазоне
примерно ±292 лет без потери точности.

Тип спроектирован с акцентом на:

- **Точность** — наносекундное разрешение без округлений
- **Компактность** — занимает ровно 8 байт (`i64`)
- **Безопасность** — поддержка checked и saturating операций
- **Совместимость** — используется вместе с [`Time`](crate::Time)

2. Создание значений

[`Duration`] можно создавать из различных единиц времени:

- `from_nanos`
- `from_micros`
- `from_millis`
- `from_seconds`
- `from_minutes`
- `from_hours`
- `from_days`

Для безопасного создания предусмотрены checked-варианты, возвращающие `None`
при переполнении.

3. Арифметика

Тип поддерживает стандартные арифметические операции:

- Сложение и вычитание (`Add`, `Sub`)
- Присваивающие операции (`AddAssign`, `SubAssign`)
- Унарный минус (`Neg`)

Также доступны:

- `checked_add`, `checked_sub` — безопасные операции с проверкой переполнения
- `saturating_add`, `saturating_sub` — операции с насыщением
- `try_add`, `try_sub` — возвращают [`GnssTimeError`] при переполнении

4. Инспекция

Методы доступа позволяют получить значение в различных единицах:

- `as_nanos`, `as_micros`, `as_millis`, `as_seconds`
- `as_seconds_f64` — с плавающей точкой

Дополнительно:

- `is_positive`, `is_negative`, `is_zero`
- `abs` — модуль (с проверкой переполнения)

5. Форматирование

Реализован [`Display`], который выводит длительность в виде:

[−]Xs Yns

 Примеры:

 - `1s 0ns`
 - `-3s 141592654ns`
- `0s 500000000ns`

6. Гарантии

- Размер типа фиксирован и равен 8 байтам
- Все операции либо безопасны, либо явно сигнализируют об ошибке
- Поведение при переполнении полностью контролируемо (checked / saturating / panic)

[`Duration`] является фундаментальным строительным блоком для всей арифметики времени
 в библиотеке.
