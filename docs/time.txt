Time<S>

Модуль определяет [`Time<S>`] — основной тип метки времени,
параметризованный системой отсчёта (`TimeScale`).

1. Общая идея

В библиотеке существует несколько временных шкал (GPS, GLONASS,
Galileo, BeiDou, UTC, TAI), каждая из которых:

- использует собственную эпоху
- может иметь смещение относительно других шкал
- в некоторых случаях зависит от leap seconds

Тип [`Time<S>`] представляет момент времени **в конкретной шкале `S`**
и хранится как количество наносекунд с начала эпохи этой шкалы.

Ключевая идея:

Time<S> — это "число + система координат времени"

За счёт параметра `S` (тип шкалы) невозможно случайно смешать разные
системы времени на уровне компиляции.

2. Внутреннее представление

u64 (наносекунды от эпохи) + PhantomData<S>

- Размер: всегда 8 байт
- Zero-cost абстракция (тип шкалы не хранится в runtime)

3. Конвертация между шкалами

Все преобразования выполняются через TAI (International Atomic Time):

T_tai = T_self + offset

Алгоритм:

S → TAI → T

Реализовано через:

- `to_tai()`
- `from_tai()`
- `try_convert()`

Важно

- Для шкал с фиксированным смещением (GPS, Galileo, BeiDou, TAI)
конвертация выполняется напрямую
- Для контекстных шкал (UTC, GLONASS)
требуется информация о leap seconds

В таких случаях возвращается:

Err(GnssTimeError::LeapSecondsRequired)

4. Арифметика

Поддерживаются операции:

- `Time + Duration`
- `Time - Duration`
- `Time - Time -> Duration`

При этом:

- операции между разными шкалами **невозможны на уровне типов**
- переполнение контролируется (`checked`, `saturating`, `try_*`)

5. Специализированные конструкторы

Для некоторых шкал добавлены удобные API:

- GPS:
  - `from_week_tow`
  - `week`,
  - `tow_seconds`

- GLONASS:
  - `from_day_tod`

Это отражает реальные форматы данных GNSS.

6. Гарантии

- Тип всегда занимает 8 байт
- Невозможно смешать шкалы времени случайно
- Все конверсии проходят через строго определённый контракт
- Контекстные шкалы явно требуют внешних данных

[`Time<S>`] является центральной абстракцией библиотеки,
объединяющей разные системы времени в единый, типобезопасный API.
