# gnss-time

**Типобезопасное время для GNSS-систем на Rust.**

`gnss-time` — это минималистичное ядро с нулевыми накладными расходами для работы
со временем в спутниковых навигационных системах (`GLONASS`, `GPS`, `Galileo`, `BeiDou`).

Это **не** GNSS-движок и **не** RTK-библиотека. Это строго типизированная временная
модель, спроектированная для предотвращения ошибок.

## Возможности

- **Типобезопасные шкалы времени** — `GLONASS`, `GPS`, `Galileo`, `BeiDou`, `TAI`,
  `UTC`
- **Фиксированные эпохи** для каждой системы (1980-01-06, 1996-01-01, и т.д.)
- **Конвертация между шкалами** через TAI как единый пивот
- **Специфичные форматы** — `Week:TOW` для GPS/Galileo/BeiDou, `Day:TOD` для GLONASS
- **Интервалы времени** (`Duration`) с наносекундной точностью и контролем переполнения
- **Zero-cost абстракции** — временные метки занимают 8 байт
- **`no_std`-совместимость** — подходит для встраиваемых систем

## Пример

```rust
use gnss_time::{Time, Duration, Gps};

let epoch = Time::<Gps>::EPOCH;
let one_week = Time::<Gps>::from_week_tow(1, 0.0).unwrap();
let diff = one_week - epoch;

assert_eq!(diff.as_seconds(), 604_800);
```

## Поддерживаемые шкалы времени

| Шкала   | Эпоха              | Формат          | Смещение относительно TAI |
| ------- | ------------------ | --------------- | ------------------------- |
| GLONASS | 1996-01-01 UTC(SU) | `GLO D:TOD.mmm` | контекстное (требует LS)  |
| GPS     | 1980-01-06 UTC     | `GPS W:TOW.mmm` | +19 с (фиксированное)     |
| Galileo | 1999-08-22 UTC     | `GAL W:TOW.mmm` | +19 с (фиксированное)     |
| BeiDou  | 2006-01-01 UTC     | `BDT W:TOW.mmm` | +33 с (фиксированное)     |
| TAI     | 1958-01-01         | `TAI +Ss Nns`   | 0 с (пивот)               |
| UTC     | 1972-01-01         | `UTC +Ss Nns`   | контекстное (требует LS)  |
